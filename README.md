# ATLAS — The AI Workflow Infrastructure

> By AIAstralis

**Google indexed the web. Atlas indexes intelligence.**

ATLAS is the operating system for discovering, comparing, and combining AI tools into production-ready workflows. Describe your goal, and ATLAS builds the system.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + Custom Cosmic UI
- **Deployment**: Vercel

## Features

- 🌌 Cinematic Cosmic UI landing page
- 🤖 AI Stack Generator with mock results
- 🔍 Tool Explorer with 12+ indexed tools
- 📚 Curated workflow stack library
- 🏷️ Category browsing
- 🔎 Tool search
- 📡 SEO-optimized (sitemap, robots, metadata)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

```bash
npm run build
# Deploy to Vercel
vercel
```
